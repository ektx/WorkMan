# vue 项目基础模板功能

## 接口基础
```javascript
{
	// 服务状态
	"success": true,
	// 状态码
	"status": 200,
	// 服务器信息
	"msg": "服务请求成功!",
	// 业务数据
	"data": {
		"name":  "Welcome Use Mock!"
	}
}
```

## 数据类型

| 名称      | 说明   | 示例                            |
| ------- | ---- | ----------------------------- |
| string  | 字符串  | {name: "zwl"}                 |
| number  | 数字   | {age: 18}                     |
| boolean | 布尔值  | {superman: t}                 |
| date    | 日期   |                               |
| null    | 空    | {title: null}                 |
| array   | 数组   | {books:['css', 'html', 'js']} |
| object  | 对象   | { people: {name: "zwl" } }    |
|         |      |                               |
